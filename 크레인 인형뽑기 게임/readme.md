# break와 continue 의 차이
  
  break 는 자신이 포함ㅁ된 가장 가까운 반복문을 벗어난다.
  
  continue는, 반복문의 끝으로 넘어가서 다음 반복으로 넘어간다.
  
  
  
# 다른 사람 코드로 부족한 부분 알기
